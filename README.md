### 앱소개

* 공유하고 싶은 메뉴(음식)를 사진과 간단한 정보들을 함께 공유하세요!
* 다른 사람들이 공유한 음식의 가격을 맞춰 보세요!
* 새로운 메뉴와 식당을 발견 해 보세요!

### 상세 소개

* vercel 호스팅과 firebase 데이터베이스를 연동해 운영 되고 있습니다.


### 추가/수정 예정
* 의견 보내기 
* 사진을 검열, 필터링 할 수 있는 기술적/법적인 장치가 없다보니 게시 요청 -> 관리자 승인 -> 게시의 과정이 필요 해보임.
